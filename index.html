VOZTG

Como TCC desenvolvi um Sistema de Comunicação para Deficiente Motor, Visual e Mudo Utilizando EEG.

Para realização deste projeto foi efetuado um estudo de caso com um usuário onde, após acidente automobilístico, ficou com sua visão, fala e funções motoras comprometidas, porém com a capacidade intelectual intacta, situação semelhante à Síndrome de Locked in (LIS).

Foi criado por parte de familiares um modo de comunicação baseado no piscar dos olhos. Funciona da seguinte forma: Um locutor anuncia ao usuário os grupos de letras, divididos entre vogal, antes de “m” ou depois de “m”; Ao chegar à sua escolha, o usuário pisca os olhos; O locutor passa então a anunciar cada letra do grupo pertencente; Ao chegar à letra escolhida o usuário pisca novamente; A letra escolhida é escrita e o processo é repetido a cada letra da palavra que o usuário deseja “pronunciar”.

O VOZTG utiliza a mesma ideia de seleção, porém sem a necessidade de um locutor físico, buscando trazer autonomia na comunicação do deficiente.

A tecnologia mais promissora para realizar a entrada de dados no sistema é a EEG, já que dispensa pouca atividade muscular. No desenvolvimento desde projeto foram utilizados os componentes: Dispositivo EEG EPOC, Sintetizador de voz Mbrola, e um editor de texto prosódico.

O software dispõe de ajustes que podem ser facilmente configurados por um usuário, como velocidade de narração, voz e quantidade de repetições.

Uma grande facilidade que o sistema oferece é o pré cadastramento de textos que podem ser depois rapidamente acessados utilizando uma chave. Dessa forma o usuário precisaria apenas escrever "aga" para conseguir dizer "Preciso de um pouco de água", por exemplo.

Na tela principal é possível, de maneira independente, selecionar as letras ou opções e ordenar ao sistema que a frase seja “falada”, além de desfazer a ultima operação e apagar o texto por completo.

O sistema reconhece duas entradas de dados: clique esquerdo e clique direito do mouse. Portanto pode ser utilizado por qualquer dispositivo capaz de emular um mouse, ou qualquer hardware projetado a partir de um.

Ao “clicar” com o botão esquerdo, o VOZTG começa a anunciar os grupos de opções ao usuário. Ao chegar ao grupo desejado, o usuário “clica” com o botão esquerdo, para realizar a seleção. O sistema começará então a anunciar as letras do grupo escolhido, uma após a outra, até a seleção ou até atingir a quantidade de repetições determinada nas configurações. A letra escolhida será escrita no campo de texto. O processo de opções começa novamente, até o usuário “falar” ou atingir a quantidade de repetições limite. Para que a frase escrita no campo de texto seja “falada”, o usuário deve “clicar” com o botão direito. Para que o usuário possa desfazer a ultima ação realizada, deve-se “clicar” com o botão esquerdo e mantê-lo pressionado por dois segundos. O VOZTG falará “DESFAZER” e o campo de texto voltará ao estado anterior. Para apagar a ultima letra escrita, o usuário deve “clicar” com o botão direito e mantê-la por dois segundos. O VOZTG falará “LETRA APAGADA” e a ultima letra escrita será apagada do campo de texto.

A primeira versão do sistema foi escrita em Java 7 e é esta a versão que foi apresentada como TCC. Atualmente a segunda versão do sistema é escrita em C#.

Atualmente, o VozTG está sendo dividido em duas partes:

Uma bilbioteca chamada Vox, responsável pela síntese;

Um aplicativo que utiliza essa biblioteca, o VozTG.


Esse código será revisado e adicionado ao GitHub, para manter a evolução do sistema.
